# test-language

A Claude Code plugin that translates user input to a randomly chosen language and provides cultural context about the country where that language is spoken.

## Features

- Randomly selects from a diverse pool of languages (common and uncommon)
- Provides phonetic pronunciation guides
- Shares interesting cultural and geographical facts about the country

## Installation

```bash
claude --plugin-dir /path/to/test-language
```

## Usage

Simply ask Claude to translate something:

- "Translate 'Good morning' to a random language"
- "Surprise me with a translation of 'The weather is beautiful today'"
- "Say 'hello world' in another language"
